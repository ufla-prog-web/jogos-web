# Jokenpô - Projeto de Desenvolvimento Web

Este é um jogo de **Jokenpô (Pedra, Papel e Tesoura)** desenvolvido como parte da disciplina de **Desenvolvimento Web** da graduação. O projeto permite que o jogador enfrente a CPU em partidas no formato "melhor de 3" ou "melhor de 5".

---

## 📁 Estrutura do Projeto

```
page/
│
├─ img/
│ ├─ icon.png
│ ├─ papel.png
│ ├─ pedra.png
│ └─ tesoura.png
│
├─ index.html
├─ script.js
└─ style.css
```

- **img/**: Contém o ícone e as imagens do jogo.  
- **index.html**: Estrutura principal do jogo.  
- **script.js**: Lógica do jogo, incluindo seleção de jogadas, placar e animações.  
- **style.css**: Estilização do jogo e efeitos visuais para as mensagens de vitória, derrota e empate.  

---

## 🎮 Como Jogar

1. Abra o arquivo `index.html` em seu navegador.  
2. Escolha sua jogada clicando em **Pedra**, **Papel** ou **Tesoura**.  
3. Clique no botão **Jogar**.  
4. O resultado aparecerá e o placar será atualizado.  
5. O jogo termina quando o jogador ou a CPU atingir a quantidade necessária de vitórias, dependendo do modo selecionado:  
   - **Melhor de 3**  
   - **Melhor de 5**  
6. Após o término, clique em **Jogar novamente** para reiniciar a partida.

---

## ⚙️ Funcionalidades

- Escolha entre **Pedra, Papel e Tesoura**.  
- Placar atualizado em tempo real.  
- Modo de jogo configurável: **Melhor de 3** ou **Melhor de 5**.  
- Mensagens animadas para vitória, derrota ou empate.
