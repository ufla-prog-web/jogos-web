# 🍦 Bad Ice Cream 

Um jogo de labirinto construído com **HTML5**, **JavaScript**, **CSS**. Inspirado no jogo "Bad Ice Cream", este projeto traz a mecânica nostálgica de coletar frutas e fugir de inimigo, mas com uma interface moderna e animações fluidas.

## 🎮 Sobre o Jogo

O objetivo é simples: limpe o labirinto coletando todas as frutas deliciosas antes que o inimigo encoste em você! Escolha o seu sabor de sorvete favorito e enfrente as fases. 

Este projeto foi desenvolvido com o objetivo de aplicar os conceitos estudados na disciplina de Programação Web da Universidade Federal de Lavras, como estruturação de páginas, estilização, manipulação do DOM, eventos e lógica de programação.


## 🕹️ Como Jogar

1. Escolha o seu sabor de sorvete clicando no menu inicial (Morango, Baunilha ou Chocolate).
2. Clique em **JOGAR**.
3. Use as **Setas do Teclado** (`Cima`, `Baixo`, `Esquerda`, `Direita`) para mover o personagem.
4. Colete todas as frutas (🍓/🍌/🍉) para passar para a próxima fase.
5. Evite o inimigo a todo custo!

## 🚀 Como Executar o Projeto

- Método 1:

Entre no link pelo navegador: https://bianca-feliciano.github.io/bad_ice_cream/ 

- Método 2:

Como o jogo utiliza apenas tecnologias nativas da web, não há necessidade de instalar dependências.

1. Faça o clone deste repositório ou baixe os arquivos.
2. Certifique-se de que todas as imagens de *sprites* (`icecream1.png`, `icecream2.png`, `icecream3.png`, `enemy.png`, `fruit.png`, `ice.png`, `wall.png`) estão na mesma pasta que os arquivos de código.
3. Abra o arquivo `index.html` em qualquer navegador moderno.
4. Divirta-se!
