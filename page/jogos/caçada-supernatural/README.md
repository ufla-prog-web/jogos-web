# Caçada Sobrenatural

## Descrição

Caçada Sobrenatural é um jogo web de investigação paranormal feito com HTML, CSS e JavaScript puro. O jogador analisa pistas de um caso, identifica a criatura responsável e define a forma correta de combater a ameaça.

## Proposta

O projeto foi pensado com estética de arquivo confidencial e diário de caça sobrenatural, equilibrando imersão visual, regras claras e navegação simples. A experiência é focada em dedução, leitura de evidências e tomada de decisão final.

## Regras

- Cada partida sorteia aleatoriamente um caso sobrenatural.
- O jogador recebe uma narrativa inicial e pistas.
- Depois, deve escolher a criatura responsável.
- Em seguida, deve selecionar a forma correta de combater a criatura.
- O jogador vence apenas se acertar criatura e forma de combate.

## Tecnologias usadas

- HTML5
- CSS3
- JavaScript puro
- LocalStorage para persistir a quantidade de casos resolvidos
