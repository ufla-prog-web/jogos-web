🐍 Snake Game

Coma a comida e cresça o máximo possível!

Pontuação: 0
Recorde: 0
Status: Pronto

Como Jogar:

  • Mova a cobra: Use as setas do teclado (↑ ↓ ← →) ou WASD
  • Objetivo: Coma a comida (quadrado vermelho) para crescer
  • Cuidado: Não colida com as paredes ou com você mesmo!
  • Velocidade: Quanto mais você come, mais rápido o jogo fica
  • Pontuação: Cada comida = 10 pontos
