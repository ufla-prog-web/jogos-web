# A Provação dos Padawans - Visao Geral

## O que e o projeto
A Provação dos Padawans e um jogo educativo para navegador em que o jogador acompanha a execucao de algoritmos classicos e responde perguntas sobre o estado das variaveis durante a execucao.

## Objetivo pedagogico
Promover compreensao real de fluxo de algoritmo, com foco em:
- leitura de iteracoes e condicoes
- diferenca entre indice e valor
- entendimento de transformacoes no array ao longo do tempo

## Escopo MVP
- Modos: Arcade e Selection
- Linguagem exibida: pseudocodigo
- Motor de execucao: JavaScript
- Sem multiplayer no MVP
- Sem persistencia entre sessoes no MVP

## Requisitos basicos
- Apenas HTML, CSS e JavaScript puro
- Minimo de 2 algoritmos implementados com snapshots
- Quiz com 4 opcoes por pergunta
- Destaque da linha ativa no visualizador de codigo
- Publicacao no GitHub Pages
