# Requisitos Basicos do Projeto

## Tecnicos
1. Usar apenas HTML, CSS e JavaScript puro.
2. Funcionar integralmente no navegador.
3. Nao usar frameworks ou bibliotecas externas.

## Funcionais (MVP)
1. Tela inicial com modos Arcade e Selection.
2. Ciclo de fase com algoritmo sorteado (Arcade) ou selecionado (Selection).
3. Execucao de algoritmo em JavaScript com snapshots.
4. Visualizador de pseudocodigo modular carregado por arquivo de dados.
5. Perguntas em pontos de pausa com 4 opcoes unicas.
6. Pontuacao por tempo no Arcade.
7. Regra de erro e game over no Arcade.
