## 🎮 Sobre o Jogo

**Memory Cosmos** é um jogo da memória com tema espacial desenvolvido com HTML, CSS e JavaScript puro. O jogador deve encontrar todos os pares de cartas cósmicas no menor número de jogadas e no menor tempo possível.

### Regras
1. Clique em uma carta para virá-la.
2. Clique em outra carta para tentar encontrar o par.
3. Se as cartas forem iguais, elas ficam descobertas. ✅
4. Se forem diferentes, elas voltam à posição inicial. ❌
5. Encontre todos os pares para vencer! 🏆

### Sistema de Pontuação
| Estrelas | Fácil (6 pares) | Médio (8 pares) | Difícil (12 pares) |
|:--------:|:---------------:|:---------------:|:-----------------:|
| ⭐⭐⭐    | ≤ 10 jogadas    | ≤ 14 jogadas    | ≤ 20 jogadas      |
| ⭐⭐      | ≤ 16 jogadas    | ≤ 22 jogadas    | ≤ 32 jogadas      |
| ⭐        | > 16 jogadas    | > 22 jogadas    | > 32 jogadas      |
