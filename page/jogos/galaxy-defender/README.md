# 🚀 Galaxy Defender

Jogo digital desenvolvido com **HTML, CSS e JavaScript puro**, executado diretamente no navegador.

## 🎮 Sobre o jogo

Galaxy Defender é um jogo no estilo _space shooter_, onde o jogador controla uma nave espacial e precisa destruir inimigos que descem pela tela.

O objetivo é sobreviver o maior tempo possível, acumulando pontos e evitando perder todas as vidas.

---

## 🕹️ Controles

- ⬅️ ➡️ ⬆️ ⬇️ → Movimentar a nave
- ␣ (Espaço) → Atirar

---

## 🎯 Regras

- Cada inimigo destruído vale **10 pontos**
- Você perde 1 vida quando:
  - Um inimigo colide com sua nave
  - Um inimigo atravessa a tela
- O jogo termina quando suas vidas chegam a **0**

---

## ⚙️ Tecnologias utilizadas

- HTML → estrutura da aplicação
- CSS → layout, animações e design
- JavaScript → lógica do jogo, eventos e manipulação do DOM

---

## 🧠 Conceitos aplicados

- Manipulação do DOM

- Animação com `requestAnimationFrame`
- Lógica de programação (estado, colisão, pontuação, game loop)
