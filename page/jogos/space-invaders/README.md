# Space_Invaders

Um clone clássico do jogo Space Invaders construído inteiramente com HTML5 Canvas e JavaScript (com foco em Programação Orientada a Objetos). O objetivo do jogo é controlar sua nave e proteger a Terra de uma frota de alienígenas que descem gradualmente pela tela. 

Destrua todos eles antes que cheguem à sua base!

### Controles
- **Setas Direita/Esquerda:** Movem a sua nave pelo cenário.
- **Barra de Espaço:** Atira nos alienígenas (e também inicia/reinicia a partida).
