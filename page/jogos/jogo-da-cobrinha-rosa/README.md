# Snake Game

Este é um projeto de jogo desenvolvido para a disciplina de Programação Web.

## 🎮 Sobre o Jogo
O objetivo é controlar a cobra, comer as frutas para crescer e evitar colidir com as paredes ou com o próprio corpo.

## 🚀 Como Jogar
- Acesse o jogo através deste link: [https://anaclaracn.github.io/Snake-Game/](https://anaclaracn.github.io/Jogo-da-Cobrinha/)
- Use as **setas do teclado** para controlar a direção.
- Pressione o botão **Reiniciar** caso perca.

## 🛠️ Tecnologias Utilizadas
- **HTML5** (Estrutura e Canvas)
- **CSS3** (Estilização e Layout)
- **JavaScript Vanilla** (Lógica do jogo e manipulação do DOM)
