# 👑 Peach Maze

Jogo de labirinto desenvolvido em HTML, CSS e JavaScript puro para a disciplina **GAC116 - Programação Web 2026/1**.

## 🎮 Como Jogar

- Use as **setas do teclado** ou **WASD** para mover a princesa
- Segure **SHIFT** enquanto se move para **correr**
- Encontre a **chave dourada** pulsando no labirinto
- Com a chave em mãos, vá até a **porta** para avançar de fase
- São **3 fases**, com labirintos progressivamente mais difíceis
- Quanto mais rápido você concluir cada fase, mais **pontos** ganha!

## 🛠 Tecnologias

- **HTML5** — estrutura das telas e canvas
- **CSS3** — animações, variáveis CSS, layout responsivo
- **JavaScript puro** — lógica de jogo, Canvas API, eventos de teclado
