#  GAC-116: Desenvolvimento Web de Jogo

Este repositório contém o projeto prático desenvolvido para a disciplina **GAC-116 — Programação WEB**. O projeto consiste em um jogo no estilo  Corredor Infinito construído com tecnologias web fundamentais.

## Sobre o Jogo
O objetivo do jogador é simples: sobreviver o maior tempo possível. 
 **Mecânica:** O personagem corre automaticamente, e o jogador deve saltar sobre os obstáculos que surgem de forma aleatória na tela.  

 **Progressão de Dificuldade:** A cada **10 obstáculos** superados, a velocidade ou a frequência dos objetos aumenta, desafiando os reflexos do player.

##  Tecnologias Utilizadas
 **HTML5:** Estruturação do jogo e do ambiente web.  

 **CSS3:** Estilização, animações de cenário e interface do usuário.  

 **JavaScript:** Lógica do jogo, detecção de colisão e controle de pontuação.
