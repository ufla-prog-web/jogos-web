# 🐍 Snake Game

Jogo da Cobrinha desenvolvido como Atividade Prática 1 da disciplina **GAC116 - Programação Web (2026/1)** da UFLA.

## Como jogar

- Acesse o jogo pelo link do GitHub Pages
- Clique em **Jogar** para começar
- Use as **setas do teclado** para mover a cobra
- Coma as frutas vermelhas para crescer e ganhar pontos
- Evite bater nas paredes ou no próprio corpo
- A cobra acelera conforme sua pontuação aumenta

## Funcionalidades

- Tela inicial com botão de início
- Pontuação em tempo real e recorde salvo entre sessões
- Detecção de colisão com paredes e corpo
- Velocidade crescente a cada 50 pontos
- Tela de game over com pontuação final e reinício

## Tecnologias

- HTML5 (`<canvas>`)
- CSS3
- JavaScript puro

## Como executar localmente

Basta abrir o arquivo `index.html` no navegador. Nenhuma instalação necessária.
