# 🎮 Entregador do Caos

## 📌 Descrição

O **Entregador do Caos** é um jogo desenvolvido utilizando HTML, CSS e JavaScript puro, onde o jogador controla um entregador que precisa coletar encomendas enquanto evita erros durante o processo.

O jogo foi criado como parte da disciplina de Programação Web.

---

## 🎯 Objetivo

O objetivo do jogo é **coletar 10 encomendas (📦)** para vencer.

---

## 🎮 Como jogar

* Use as teclas:

  * ⬅️ (seta esquerda) para mover para a esquerda
  * ➡️ (seta direita) para mover para a direita

* O personagem deve:

  * ✅ Coletar encomendas (📦) → aumenta a pontuação
  * ❌ Evitar bombas (💣) → diminui a pontuação

---

## 🧠 Regras do jogo

### ✅ Pontuação

* Cada 📦 coletada: **+1 ponto**
* Cada 💣 coletada: **-2 pontos**

---

### ❌ Condições de derrota

O jogador perde se:

* A pontuação for **menor ou igual a -5**
* Deixar cair **mais de 3 encomendas (📦)**

---

### 🏆 Condição de vitória

* O jogador vence ao atingir **10 pontos**

---

## ⏸️ Funcionalidades

* Movimento com teclado
* Sistema de pontuação
* Sistema de vidas (encomendas perdidas)
* Condição de vitória e derrota
* Reinício automático ao fim do jogo
* Botão de pausa/continuação
* Interface visual estilizada

---
