# Tetris em JavaScript

Este projeto é um jogo Tetris feito com JavaScript, HTML e CSS puros.

## Como rodar

1. Baixe ou clone este repositório.
2. Abra o arquivo `index.html` no navegador.

## Controles

- Seta para a esquerda: mover para a esquerda
- Seta para a direita: mover para a direita
- Seta para baixo: queda suave
- Seta para cima ou X: girar peça
- Espaço: queda rápida
- P: pausar ou retomar
- R: reiniciar
