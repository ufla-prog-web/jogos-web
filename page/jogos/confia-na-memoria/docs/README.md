# Confia na Memória

Jogo de padrões de cores desenvolvido com HTML, CSS e JavaScript puro, sem dependências externas.

## Como Jogar

1. Selecione um nível de dificuldade (Fácil, Médio, Difícil ou Infinito)
2. Clique em "Começar" para iniciar
3. Observe a sequência de cores que aparece
4. Repita a sequência clicando nos botões na mesma ordem
5. Cada rodada adiciona uma cor à sequência
6. Complete todas as rodadas do seu nível para vencer

## Dificuldades

- Fácil: 5 rodadas para vencer
- Médio: 10 rodadas para vencer
- Difícil: 15 rodadas para vencer
- Infinito: sem limite de rodadas

## Características

- Código puro: HTML, CSS, JavaScript
- Interface minimalista e moderna
- Sons gerados com Web Audio API
- Recorde persistente (localStorage)
- Responsivo para dispositivos móveis
- 8 cores interativas em grid 4x2

## Tecnologias

- HTML5 (estrutura semântica)
- CSS3 (estilos e animações)
- JavaScript ES6+ (lógica e interatividade)
- Web Audio API (geração de sons)
