# Jogo da Memória 🃏

Jogo da memória com baralho completo (52 cartas), desenvolvido em HTML, CSS e JavaScript puro.

## 🎮 Como jogar

### 📥 Instalação
1. Clone o repositório:
```bash
   git clone https://github.com/Thiagoferreira13/jogo-da-memoria.git
```
2. Acesse a pasta do projeto

3. Abra o arquivo `index.html` no navegador

### 🃏 Regras
- Escolha a dificuldade: **Fácil** (6 cartas), **Médio** (12 cartas) ou **Difícil** (24 cartas)
- Clique em duas cartas por vez para virá-las
- Se formarem um par ✅, ficam reveladas
- Se não formarem ❌, voltam ao verso após 1 segundo
- Vença encontrando **todos os pares** no menor tempo e movimentos possível!
## 🕹️ Funcionalidades

- Baralho completo com 52 cartas únicas (4 naipes × 13 valores)
- Cartas sorteadas aleatoriamente a cada partida
- Contador de movimentos, tempo e pares encontrados
- 3 níveis de dificuldade
- Mensagem de vitória ao completar o jogo

## 🛠️ Tecnologias

- HTML5
- CSS3
- JavaScript (puro)
