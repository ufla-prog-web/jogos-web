# 🍓 Memória das Frutas

Jogo da memória desenvolvido para a disciplina **GAC116 - Programação Web (2026/1)** da UFLA.

## 🎯 Objetivo

Encontrar todos os pares de frutas no menor número de jogadas e no menor tempo possível.

## 🕹️ Como jogar

- Clique em duas cartas para revelar os símbolos.
- Se os símbolos forem iguais, o par permanece aberto.
- Se forem diferentes, as cartas viram novamente.
- O jogo termina quando todos os pares forem encontrados.

## ✨ Funcionalidades

- Interface visual moderna e responsiva  
- Contador de jogadas  
- Contador de pares encontrados  
- Cronômetro em tempo real  
- Recorde salvo no navegador (LocalStorage)  
- Botão de reiniciar jogo  

## 🛠️ Tecnologias utilizadas

- HTML5  
- CSS3  
- JavaScript
